# dotfiles

**Set sane defaults on macOS**

```
./configure-macos.sh
```

**Install homebrew**

```
./install-brew.sh
```

**Install asdf-vm (primarily for elixir and erlang)**

```
./install-asdf.sh
```

**Install fisherman**

```
./install-fisherman.sh
```

**Install spacemacs**

```
./install-spacemacs.sh
```
